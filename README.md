Геометрия - прямоугольники

Задание

Написать класс Rectangle, задающий прямоугольник с указанными координатами левого верхнего угла, шириной и высотой, а также свойствами, позволяющими узнать периметр и площадь прямоугольника. Обеспечить нахождение объекта в заведомо корректном состоянии (отрицательные значения недопустимы). Написать программу, демонстрирующую использование данного прямоугольника.

Стек технологий:

C# / .NET Консольное приложение без сторонних библиотек

Функциональность:

1. Возможность задавать ширину, высоту и координаты точки левого верхнего угла прямоугольнки.
2. Проверка на правильность заданных данных.

Структура проекта:

Rectangle.cs - основной класс программы
Program.cs - точка входа в программу

Пример работы программы:

==Прямоугольник==

Площадь прямоугольника: 100

Периметр прямоугольника: 40

Координата точки X: 2

Координата точки Y: 3


==Изменение значений==

==Прямоугольник==

Площадь прямоугольника: 320

Периметр прямоугольника: 72

Координата точки X: -5

Координата точки Y: 0
